# Промпт для редизайна сайта Seller Hub в Google Sites

## Контекст

Ниже — детальный промпт для AI-инструмента или дизайнера, чтобы воссоздать и улучшить визуальный стиль платформы **Seller Hub** в Google Sites. Стиль — editorial / журнальный минимализм.

---

## Промпт

```
Создай современный сайт в Google Sites в стиле «editorial minimalism» для SaaS-платформы управления продажами на маркетплейсах (Wildberries, Ozon, Яндекс Маркет) под названием «Seller Hub».

### Общий стиль и философия дизайна

Стиль — журнальный минимализм (editorial design):
- Чистота и воздух: много белого пространства, никаких теней и градиентов
- Типографика как главный инструмент дизайна
- Минимум декора: тонкие линии-разделители (1px, цвет #e8e5e1), никаких иконок ради иконок
- Ощущение премиальности через сдержанность, а не через «блеск»
- Вдохновение: Stripe, Linear, Notion, Apple — но без копирования

### Цветовая палитра

Строго придерживайся этих цветов:
- Фон страницы: #faf9f7 (тёплый off-white)
- Фон карточек/блоков: #ffffff (чистый белый)
- Тёмные блоки (hero, sidebar): #0a0a0a (почти чёрный)
- Основной текст: #1a1a1a
- Вторичный текст: #6b6b6b
- Приглушённый текст (подписи, лейблы): #9a9a9a
- Акцент (терракота): #c45d3e — используется ТОЛЬКО для:
  — логотипа
  — активных элементов навигации
  — CTA-кнопок (hover-состояние)
  — ключевых цифр/метрик
- Светлый акцент (для фона бейджей): #f0e0da
- Границы: #e8e5e1 (тёплый серый)

### Типографика

- Заголовки (H1, H2): шрифт с засечками, стиль italic. Если доступен — «Instrument Serif». Альтернатива — Georgia italic. Жирность: normal (400), НЕ bold.
- Основной текст: Inter или системный sans-serif. Размер 14px, line-height 1.5.
- Лейблы/подписи: 11px, uppercase, letter-spacing 0.08–0.1em, font-weight 600, цвет #9a9a9a.
- Крупные цифры (метрики): шрифт с засечками (как заголовки), размер 28–36px.

### Структура страниц

#### 1. Главная страница (лендинг)
Сверху вниз:

**Hero-секция:**
- Фон: #0a0a0a (тёмный)
- Логотип: квадрат 36×36px с закруглёнными углами (8px), фон #c45d3e, буква «S» белым шрифтом с засечками, italic
- Рядом с логотипом: «Seller Hub» — шрифт с засечками, italic, белый
- Заголовок: крупный (48–52px), шрифт с засечками, italic, белый. Текст: «Управляйте продажами эффективно»
- Подзаголовок: 15px, цвет #737373. Текст: «Единая платформа для аналитики, ценообразования и управления товарами на маркетплейсах»
- CTA-кнопка: фон #1a1a1a → на hover #333, текст белый, скруглённые углы 6px, без тени
- Внизу hero — 3 метрики в ряд, разделённые вертикальной линией (#2a2a2a):
  — «3 маркетплейса» / «24/7 мониторинг» / «AI обогащение»

**Секция возможностей:**
- Лейбл секции: «ВОЗМОЖНОСТИ» — 11px uppercase, #9a9a9a, с горизонтальной линией-разделителем справа
- Сетка 3×2 или 2×3 карточек:
  — Карточка: белый фон, border 1px solid #e8e5e1, border-radius 8px, padding 24px
  — Заголовок карточки: шрифт с засечками, 18–20px
  — Описание: 14px, цвет #6b6b6b
  — Иконка: минималистичная, stroke-only (не filled), цвет #1a1a1a

Модули для карточек:
1. **Авто-импорт** — Парсинг каталогов поставщиков, автоматический импорт товаров с AI-классификацией
2. **Ценообразование** — Расчёт цен на основе закупочной стоимости, логистики и желаемой маржи
3. **AI-обогащение** — Генерация заголовков, описаний и характеристик для карточек товаров
4. **Мониторинг цен** — Безопасное изменение цен с откатом и защитой от демпинга
5. **Аналитика** — Финансы, отмены, региональная статистика, sentiment-анализ отзывов
6. **AI-агенты** — Автономные агенты для обогащения, классификации и мониторинга

**Секция «Как это работает»:**
- 3–4 шага горизонтально
- Каждый шаг: номер крупным шрифтом с засечками (терракота #c45d3e), заголовок, короткое описание
- Шаги соединены тонкой пунктирной линией

**Секция статистики/доверия:**
- Сетка 4 колонки, разделённые 1px линией (#e8e5e1)
- Формат каждой ячейки: лейбл сверху (11px uppercase #9a9a9a), цифра снизу (28px, шрифт с засечками)
- Пример метрик: «10K+ товаров» / «50+ продавцов» / «99.9% uptime» / «< 2 сек отклик»

**Footer:**
- Фон: #0a0a0a
- Логотип + копирайт: «© 2026 Seller Hub»
- Ссылки мелким шрифтом, цвет #555

#### 2. Страница «Возможности» (подробная)
- Hero: тёмный фон, заголовок «Всё для управления маркетплейсами»
- Далее — чередование секций (зигзаг): текст слева / скриншот справа, и наоборот
- Каждая секция посвящена одному модулю (Авто-импорт, Цены, AI, Аналитика и т.д.)
- Скриншоты оформить в мокап с тонкой рамкой #e8e5e1 и скруглёнными углами

#### 3. Страница «О платформе»
- Минималистичная, текстовая
- Заголовок с засечками, italic
- Текст в одну колонку, max-width ~700px по центру
- Миссия, история, технологии (Flask, Python, AI/GPT, Wildberries API)

#### 4. Страница «Контакты»
- Простая форма: Имя, Email, Сообщение
- Поля: border 1px solid #e8e5e1, border-radius 6px, при фокусе — border-color #c45d3e
- Кнопка отправки: фон #1a1a1a, текст белый

### Ключевые правила оформления

1. **Никаких теней** — вместо box-shadow используй border 1px solid #e8e5e1
2. **Никаких градиентов** в карточках — только solid colors
3. **Border-radius**: максимум 8px для карточек, 6px для кнопок и полей ввода
4. **Кнопки**:
   - Primary: фон #1a1a1a, текст белый
   - Secondary: border 1px solid #1a1a1a, текст #1a1a1a, прозрачный фон
   - Accent (редко): фон #c45d3e, текст белый
5. **Бейджи/теги**: скруглённые (rounded-full), пастельный фон + насыщенный текст того же цвета
6. **Таблицы** (если есть): заголовки — 11px uppercase #9a9a9a, строки с hover-подсветкой #faf9f7
7. **Изображения**: если используются — в стиле flat illustration или скриншоты в мокапах с тонкой рамкой

### Адаптивность

- На мобильных: сетка карточек в 1 колонку, hero-заголовок уменьшается до 32px
- Метрики — 2 колонки на мобильных
- Навигация: простое гамбургер-меню

### Тон текста

- Профессиональный, но не сухой
- Русский язык
- Короткие фразы, конкретика вместо воды
- Примеры хорошего текста:
  — «Ценообразование на автопилоте» вместо «Автоматизированная система ценообразования»
  — «Импорт товаров в один клик» вместо «Удобный функционал импорта товаров»

### Чего НЕ делать

- Не добавлять стоковые фото людей
- Не использовать яркие/кислотные цвета
- Не делать скруглённые углы больше 8px
- Не использовать тени (box-shadow)
- Не добавлять анимации — Google Sites их не поддерживает
- Не перегружать страницу: лучше меньше блоков, но качественных
- Не использовать шрифты bold для заголовков — только regular (400) italic
```

## Дополнительные советы для Google Sites

1. **Шрифты**: Google Sites ограничен в шрифтах. Используй «Roboto Serif» для заголовков (ближайший к Instrument Serif) и «Roboto» для основного текста (ближайший к Inter).
2. **Тёмные секции**: используй фоновые изображения solid-color (#0a0a0a) для hero и footer.
3. **Карточки**: создавай через «Text box» с настроенными отступами, или используй встроенные карточки Google Sites с минимальным оформлением.
4. **Сетка**: Google Sites поддерживает 1–4 колонки — используй это для раскладки карточек и метрик.
5. **Иконки**: вставляй как изображения (SVG/PNG) — stroke-style иконки из Heroicons или Lucide.
6. **Разделители**: используй встроенный элемент «Divider» для горизонтальных линий.
