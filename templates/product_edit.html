Важно о Wildberries API

Изменения отправляются в Wildberries API:

  • Модерация может занять от нескольких минут до нескольких часов
  • Некоторые поля могут быть защищены от редактирования в зависимости от категории
  • Если получаете ошибку 404 - эндпоинт API может быть недоступен или требуется другой формат запроса
  • WB API v2 может требовать полную карточку вместо частичных изменений
