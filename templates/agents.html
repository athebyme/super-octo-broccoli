Как работают агенты?

Агенты — это AI-сервисы, которые автоматизируют рутину: категоризация, SEO, проверка карточек, импорт. Каждый агент работает автономно, берёт задачи из очереди и показывает ход выполнения в реальном времени.

1 Активируйте агента

Нажмите «Активировать» на карточке нужного агента. Система зарегистрирует его и выдаст API-ключ. {% if not current_user.is_admin %} Доступно только администраторам. {% endif %}

2 Запустите процесс

Запустите контейнер агента (Docker) с полученным API-ключом. Агент подключится к платформе и перейдёт в статус «Online».

3 Создайте задачу

Задачи создаются автоматически (при импорте товаров) или вручную через API. Агент возьмёт задачу из очереди и выполнит её.
