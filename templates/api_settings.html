Как получить API ключ?

  1. Зайдите в личный кабинет Wildberries
  2. Перейдите в Профиль → Настройки → Доступ к API
  3. Выберите тип токена "Content" для работы с карточками товаров
  4. Нажмите "Создать токен"
  5. Скопируйте токен (он показывается только один раз!)
  6. Вставьте токен в форму ниже
