Как работает Smart Parser

  • Smart Parse — определяет бренды через BrandEngine (fuzzy matching по реестру), маппит категории WB, извлекает размеры/материалы/цвета из названий
  • Валидация брендов — проверяет каждый бренд поставщика по списку брендов WB, ищет ближайшее совпадение
  • Валидация характеристик — проверяет AI-данные по справочнику WB для конкретной категории, автокорректирует опечатки и регистр
  • Все операции алгоритмические — не тратят токены AI
